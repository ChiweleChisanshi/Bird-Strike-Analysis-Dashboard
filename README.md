# Bird-Strike-Analysis-Dashboard
This project is a Tableau dashboard that analyzes birdstrike incidents using aviation wildlife strike data from 1990 to 2020.
Birdstrike Analysis Dashboard
Overview

This project is a Tableau dashboard that analyzes birdstrike incidents using aviation wildlife strike data from 1990 to 2020.

The goal of the project was to explore patterns in birdstrike incidents, including where they happen, when they happen, the cost of repairs, injury impact, wildlife population size, and differences across states, operators, and time periods.

Project Goal

The main goal was to create visualizations that help answer questions such as:

Which states have the highest number of birdstrikes?
How do birdstrikes change over time?
Which operators experience the highest repair costs?
What is the average number of injuries related to birdstrike incidents?
How do monthly birdstrike patterns compare between states?
How does wildlife population size relate to birdstrike incidents?

Key Insights

Some of the main insights explored in this project include:

Birdstrike incidents vary significantly by state.
Some states show higher monthly birdstrike activity than others.
Repair costs can differ widely depending on the operator and incident type.
Injury counts are generally low, but still important for safety analysis.
Wildlife population size and location can help provide context for birdstrike frequency.
Tableau visualizations make it easier to compare trends across time, geography, and operators.

What I Learned

This project helped me practice using Tableau to analyze a large dataset and communicate findings through visualizations. I learned how to compare incidents by state, evaluate repair costs, and look for patterns across time and location.

I also learned how important it is to design dashboards that make complex data easier to understand. Birdstrike data includes many different fields, so choosing the right visuals was important for making the analysis clear.

Conclusion

This project shows how Tableau can be used to analyze aviation safety data and identify patterns in birdstrike incidents. The dashboard helps summarize incident frequency, repair costs, injury impact, state-level differences, and time-based trends.
